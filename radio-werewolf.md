# 無線人狼進行マニュアル

これは特小無線を使った人狼です。
特小無線ならではのルールになっています。
GMとの連絡とマップ状況把握用にスマートフォンを使う必要がありそう。

## 1. 基本設定・配置ルール
* 非対面 
  * プレイヤー同士は一切顔を合わせず、街中に潜む一般人に紛れて特小無線での交信のみで進行します
* ロケーション
  * あらかじめ地図上に指定された、互いに**50m以上離れた複数のポイント**のいずれかに各自が配置されます
  * ポイントの総数は**参加人数 + 1箇所**用意されます
* 初期配置と役職決定
  * 初回通信（スマホ等の別手段）にて各自の初期ポイントと役職が個別に通達されます
* ランダム投票
  * 毎朝、昼の投票の順番はランダムに決定され、アナウンスされます
* 不感地帯（通信不能）の許容
  * 配置や建物の遮蔽物によって、**特定のメンバー同士の電波が直接届かないという状況がルール上発生します**
    * 不法局と妨害局の間であっても例外ではありません
    * 運営はあえてこのような状況を作るよう配置してください
  * 電波が届かない場合は、間にいるプレイヤーが通信を中継する必要があります。

## 2. 勝利条件
* **不法局（人狼）陣営の勝利：**
  * 「通信士陣営（生存者）」と「不法局（生存者）」の人数が同数になった時点で勝利
* **通信士陣営の勝利：** 
  * すべての「不法局」を追放（処刑投票）できれば勝利

## 3. 役職一覧

### 不法局陣営（人狼側）

#### 不法局（ふほうきょく） ［人狼］
* 電波を不法に占拠し、ゲームの崩壊を狙う黒幕。
* 毎夜、通信士陣営のプレイヤーを1名指定して「通信不能（排除）」にする。

#### 妨害局（ぼうがいきょく） ［狂人］
* 不法局の強力なサポーター。最初から誰が不法局かを知っている。
* **能力（ジャミング）** 
  * 他のプレイヤーが無線で発言している最中に、意図的に約10秒間の無変調信号（キャリア）を被せて送信し、会話を強制的に妨害できる。
  * 昼ターン毎に3回まで利用可能
* **移動能力**
  * 移動局と同様に、毎ターン（毎朝）空いているポイントへ自由に移動できる。不法局を援護するために近づくか、あえて遠くから妨害するかは自由

#### 偽装局（ぎそうきょく） ［狂人 / デコイ］
* 一般局のフリをして偽の情報を流し、場を混乱させる存在。不法局が誰かは知らない。
* **能力（混信・道連れ）**
  * 昼の投票で自分が追放されたとき、最後に強力な混信を引き起こし、生存者の中から1名を道連れにして「通信不能（排除）」にする

### 通信士陣営（市民側）

#### 固定局（こていきょく） ［市民］
* 最初から最後まで、自身の初期位置から一歩も動けない通信士
* 位置が固定されているため、「自分の耳に届く各プレイヤーの電波の強弱」が常に正確な基準となる

#### 移動局（いどうきょく） ［市民］
* マップ内をアクティブに動き回る、市民陣営の情報屋。
* **移動能力**
  * 毎朝、空いているポイント（最初からの空き地、または死亡したプレイヤーがいた場所）へ自由に移動できる。電波の届かない不感地帯を解消するための中継役などを担う

#### 監査官（かんさかん） ［占い師］
* 不法な電波を監視する、市民側のキーマン。
* **能力（スキャン）**
  * 毎夜、指定したプレイヤー1名の電波を傍受し、その正体が「不法局」か「それ以外」かを知ることができる

## 4. ゲーム進行の流れ

### ☀️ 【朝・昼のターン（表チャンネル：例4ch）】
1. **移動の完了：** 移動局・妨害局は、前夜に指定した新しいポイントへ朝までに静かに移動を完了しておく。
2. **状況アナウンス：** ランダムな投票順、および「昨晩通信不能になったプレイヤー（＝新しい空きポイント）」がアナウンスされる。
3. **議論（通信）：** 全員で情報を共有。「移動局の私、今〇番にいますが、ここからだとAさんの声がクリアに聞こえます！逆にBさんの声は聞こえません」といった位置情報を元に推理を展開する。
4. **投票・追放：** 朝アナウンスされた順番に投票を行い、最も票を集めた1名を追放する。※追放されたのが「偽装局」だった場合、その場で道連れにする1名を指名する。

### 🌙 【夜のターン（裏チャンネル:例6ch）】
1. **監査官の行動** 
  * 占いたい相手をGMに伝え、「不法局」か「それ以外」かの判定を受け取る
2. **不法局・妨害局の行動** 
  * 不法局は襲撃先をGMに伝える
  * 妨害局は（電波が届く範囲にいれば不法局と作戦を合わせつつ）次の行動に備える
3. **移動の申告（移動局・妨害局）** 
  * 次の朝に自分が移動したいポイントをGMに申告する
  * 移動能力を持つ人同士が同じポイントを選んでしまった場合は、移動は無しとなる
